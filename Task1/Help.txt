How to Install PHP in windows
Step 1: Go to official site of PHP https://www.php.net/downloads.php
Step 2: If downloading for Windows machine look for the Windows downloads link in each version of Php.
Step 3: Extract the files from downloaded zip folder.
Step 4: Move extracted folder to C drive
Step 5: Add C:\php to the path environment variable
